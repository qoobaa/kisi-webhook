# Kisi Webhook

This is a simple app that allows you to generate webhook handlers for
[Kisi](http://getkisi.com/), to deliver email messages based on
various Kisi events.

## Demo

(https://kisi-webhook.herokuapp.com/)[https://kisi-webhook.herokuapp.com/]

## Requirements

- Ruby 2.4.2
- PostgreSQL 10.1

## Development

```
bundle install
rails db:setup
rails server
```
